# Discord-tokenchecker
Discordのユーザーtokenの生存確認ができるHTML

結構前に作ったやつだから汚いね()<br>
なんで背景黒で光らせてるのかは謎。見づらい((

### 使い方
1: **Enter discord tokens**って書いてあるテキストエリアにtokenを入力(複数可)<br>
2: **VaildTokens**ってとこに表示されたやつは有効、**inVaildTokens**ってとこに表示されたやつは無効です<br>

そんだけ。おわり
